- My Work -

HeathBridge

HealthBridge is a personal health dashboard app that syncs Apple Health data from your iOS app to a backend, stores it in Postgres, and visualizes trends in a web dashboard (overview, activity, vitals, sleep, and hearing metrics). You can access it by clicking on the monitor.

Skills: Swift + HealthKit, Node.js, Express, PostgreSQL, React

Thinkboard

ThinkBoard is a full-stack MERN note app that lets users create, view, update, and delete notes.

Skills: Node.js, Express, MongoDB, React, Tailwind CSS

Natour

Natours is a web app where people can browse adventure tours, see where each trip goes, read reviews from other travelers, book a spot, and manage their account and payments in one place.

Skills: Node.js, Express, MongoDB/Mongoose, REST APIs, JWT auth, Pug SSR, Stripe payments, Mapbox, security middleware

- About me -
